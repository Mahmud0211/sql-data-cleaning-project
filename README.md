# SQL Data Cleaning Project

## 📊 Overview
This project focuses on cleaning a layoffs dataset using SQL.

## 🧹 Data Cleaning Steps
- Removed duplicates using ROW_NUMBER()
- Standardized company and country names
- Handled null and blank values
- Converted date column to proper DATE format

## 🛠 Tools Used
- MySQL
- SQL

## 📁 Project Structure
- cleaning_data.project.sql – main SQL script
- README.md – project description

## 🎯 Goal
Prepare raw data for further analysis (EDA and visualization)
